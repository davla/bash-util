getterSetter(prop, valueopt) → {jQuery|*}

Gets the value of a property or sets it if one is passed.

Prototype of the typical jQuery-style property accessor: if a value is provided, then it acts as a setter and returns the jQuery instance; if only property name is given, it acts as a getter and returns the value of the property.
Parameters:
Name Type Attributes Description
prop integer | string Index of the property to act with.
value * <optional>
 The value the property will be set to.
Returns:
This jQuery instance when used as setter, the property value when used as getter.
Type
jQuery | *
