Function footer

Prints the footer.

This function prints the footer, that is almost the same for any page. It consists in a bunch of sections:

GitHub source link and version
Link to GitHub repository for this website and current version.
Powered by
websites of software and hardware used in the web page.
Author
Name and email of autor of the web page.
License
License under which the web page is released.

Websites displayed in "Powered by" section are those passed on as arguments, from the third one onwards: these should be valid indexes of the JSON object in footer.json file. By now, however, Raspberry, Apache, MySQL and PHP are always displayed.
