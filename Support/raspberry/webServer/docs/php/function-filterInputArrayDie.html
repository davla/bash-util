Function filterInputArrayDie

Filters an input array, dying on failures.

This function filters an input array using filter_input_array() and checks for failures, printing an error message for each unsuccessful filter and dying afterwards when any is found.

In the case of every filter succeeding, the result of filter_input_array() with $add_empty argument unset is returned: that is, an array holding results, without keys for missing corresponding ones in input.

There is also the possibility to automatically add FILTER_NULL_ON_FAILURE flag to all FILTER_VALIDATE_BOOLEAN filters, by setting $strictBool to true.

See: filter_input_array()
