Function array_group

Groups rows of a bidimensional array having the same value of a cell, using this as key for the group in the returned array

This function takes as input a bidimensional array and performs a task somewhat similar to SQL group by: groups in an array the rows having the same value of $column cell and uses it as key for the rows group in the result array.

Note that $column cell is removed from groups in the result array

Located at common.php
Parameters summary
mixed[] $array Bidimensional array from which groups are pulled
string|integer $column Index of the column the input will be grouped by
Return value summary
mixed[] Keys are distinct values of $column cell; values are arrays of rows that share the same value of $column cell, but without it
