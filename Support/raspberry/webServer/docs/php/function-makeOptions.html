Function makeOptions

Converts a PHP array to HTML option tags, returning them as a string.

This function returns a string of HTML option tags, using $options elements to create them: these can be either arrays or anything castable to a string. In the former case, every key-value pair is turned to an HTML attribute-value pair, provided its value is set; the text of the option tag is the value of the special key 'text'. In the latter case, the element itself is used as text for the option tag, and the key may be used as value for 'value' HTML property, depending on $keysAsValues being set.
