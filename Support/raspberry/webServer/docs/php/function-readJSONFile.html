Function readJSONFile

Reads an associative array from a JSON file.

This function returns the associative array represented in a JSON file. If $toHTMLEntities is true, all special characters but quotes in the file will be converted to HTML entities.

Located at common.php
Parameters summary
string $filePath The path of the JSON file.
boolean $toHTMLEntities = true If true, all special characters but quote will be converted into HTML entities.
Return value summary
mixed[] The associative array represented by the JSON file.
